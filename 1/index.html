Giải Phương Trình

Bài 30

a)

\(\frac{1}{x-2} + 3 = \frac{x-3}{2-x}\)

Trước tiên, nhận thấy \(\frac{x-3}{2-x} = \frac{-(x-3)}{x-2} = -\frac{x-3}{x-2}\).

Do đó, phương trình trở thành:

\(\frac{1}{x-2} + 3 = -\frac{x-3}{x-2}\)

Nhân cả hai vế với \(x-2\):

\(1 + 3(x-2) = -(x-3)\)

\(1 + 3x - 6 = -x + 3\)

\(3x - x = 3 + 6 + 1\)

\(4x = 10\)

\(x = \frac{5}{2}\)

Kiểm tra lại điều kiện:

\(x \neq 2\)

Vậy, nghiệm của phương trình là \(x = \frac{5}{2}\).

b)

\(2x - \frac{2x^2}{x+3} = \frac{4x}{x+3} + \frac{2}{7}\)

Đưa về mẫu số chung \(x+3\):

\(2x(x+3) - 2x^2 = 4x + \frac{2(x+3)}{7}\)

\(2x^2 + 6x - 2x^2 = 4x + \frac{2x + 6}{7}\)

\(6x = 4x + \frac{2x + 6}{7}\)

Giải tiếp để tìm \(x\).

c)

\(\frac{x+1}{x-1} - \frac{x-1}{x+1} = \frac{4}{x^2-1}\)

Giải tiếp để tìm \(x\).
